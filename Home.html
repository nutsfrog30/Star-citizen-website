Star Citizen. Why you should even play!?

Star Citizen is a free-to-play video game because all money is from crowd funding. Everything is on one big massive server. The developers are also releasing a single player version of the game. The game is not yet realesed, but it is scheduled for release in 2016. This game is a MMO about space exploraton, first person space combat, etc and has some features from Freelance. Freelance is a really old game which was made by the same developer as Star Citizen, Chris Roberts.
